Web app para tracking de entrenamiento físico. Registra ejercicios, sigue tu progreso y alcanza tus metas.

## ✨ Características

- 📝 Registro de rutinas y ejercicios
- 📊 Progreso semanal con gráficos
- 🎯 Sistema de experiencia (XP) y niveles
- 👥 Perfiles de usuario
- 📱 Diseño responsive

## 🚀 Instalación

```bash
pip install -r requirements.txt
python app.py
```
